# base-converter
Overview

A C-language command-line tool for converting between any number bases (2-16), inspired by GeeksforGeeks' number system conversion principles. Designed for students to simplify course-related conversion tasks and visualize calculation processes.

Key Features

Supports bidirectional conversion between 2-16 bases (common + custom bases)
User-friendly hierarchical menu UI
Step-by-step conversion visualization
Input validation and error handling

Usage

Select "Base Conversion" from the main menu
Choose source and target bases (2-16)
Enter a valid value (0-9, A-F for hexadecimal)
View result and conversion steps
